Site Plan: "Grigio's Pharmacy"

Site Purpose

The purpose of this site is to train the skills learned in the WDD 231 course and to provide inspiration for a website for selling medicines.

Target Audience

The target audience for this site includes individuals looking to purchase medicines online, healthcare professionals seeking reliable sources for pharmaceuticals, and students or professionals interested in web development and design.

Scenarios

The main page, titled "Products", retrieves products from a JSON file and displays them. It also features a search bar that filters products by name and presents them in a grid format.

The main page is mobile-friendly, showing one product per row on smaller screens.

The "Cart" page displays the total cost of selected products and lists the items chosen for purchase.

The "About Us" page shares the pharmacy's history and introduces team members. It also includes a form for subscribing to the pharmacy newsletter.
